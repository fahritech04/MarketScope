# Scraping Policy (MVP)

1. Hanya target halaman publik tanpa login/paywall.
2. Tidak bypass CAPTCHA atau mekanisme proteksi ilegal.
3. Cek dan hormati aturan `robots.txt` sebelum scraping URL sumber.
4. Gunakan user-agent yang jelas dan transparan.
5. Terapkan delay request untuk menghindari beban berlebih.
6. Gunakan data hanya untuk analitik internal yang sah.
7. Jalankan dengan verifikasi SSL aktif sebagai default production (`ALLOW_INSECURE_SSL=false`).
